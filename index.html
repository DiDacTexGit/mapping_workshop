Web Mapping Workshop ReHash

Welcome to the Web Mapping Workshop Rehash! Learn to make useful web maps
on your RDT&E machine.

The original workshop was a joint effort between the Code for Dayton, Girl Develop It Dayton, and Dayton Data Visualization groups. Given on Sept. 10, 2016 by Michael Bowman.

Prerequisites

This workshop is beginner friendly, however there are some tools and technologies you should have a passing familiarity with before the workshop.

You will be making maps for use on web pages, so you should be familiar with the basics of writing a web page with HTML, CSS and JavaScript.

NOTE: I was able to follow without much knowledge. You can go far with the examples by cutting and pasting with with a pinch of trial and error. See Further Explorations section

A good introduction to each of these is Mozilla's Web Guides for HTML, JavaScript and CSS.

All of the maps will be made using JavaScript. Tutorial goes over everything you'll need to make a map.

And finally you will need:

  • Text editor like Sublime Text or Atom (NOTE these are not approved on RDT&E however Notepad is approved) for writing code.
  • Python
  • Chrome
